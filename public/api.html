Overview

This API provides a powerful and easy-to-use endpoint for removing backgrounds from images using AI technology. The service accepts various image formats and returns a PNG image with a transparent background.

Endpoints

POST /api/remove-background

Remove Background from Image

Removes the background from the provided image using AI processing.

Request

Content-Type: multipart/form-data

Parameters

Name Type Required Description
image File Yes The image file to process (JPEG, PNG, or WebP)

Response

Status Content-Type Description
200 OK image/png Processed image with transparent background
400 Bad Request application/json Invalid input or missing image
500 Internal Server Error application/json Server processing error

Example Request

Using curl:

curl -X POST -F "image=@path/to/your/image.jpg" \
    http://localhost:3000/api/remove-background \
    -o output.png

Using JavaScript Fetch:

const formData = new FormData();
formData.append('image', imageFile);

fetch('http://localhost:3000/api/remove-background', {
    method: 'POST',
    body: formData
})
.then(response => response.blob())
.then(blob => {
    // Handle the processed image blob
    const url = URL.createObjectURL(blob);
    // Use the URL as needed (e.g., display or download)
});
Note: The processed image will be returned as a PNG file with a transparent background, regardless of the input format. The original filename will be preserved with a .png extension.

Rate Limiting

Currently, there are no strict rate limits in place. However, please be mindful of server resources and implement appropriate throttling in your applications.

Error Responses

Error responses will be returned in JSON format:

{
    "error": "Error message describing what went wrong"
}
